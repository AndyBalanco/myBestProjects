ТИПЫ ДАННЫХ:
Примитивные типы.
1. number - используется для целых и дробных чисел (var 5)
Infinity - бесконечность  (alert( 1 / 0 );)
NaN - ошибка вычислений  (alert( "нечисло" * 2 );)
2. string - Запсываются  в двойных или одинарных кавычках (var str = "строка";)
3. boolean - Булевый(логический) тип имеет два значения true/false
4. null - имеет одно значение null (var age = null;) меет смысл "ничего" или "значение не известно"
5. undefined - имеет смысл "значение не присвоено"
Тип Объекты:
6. object - служит для колекции данных и для обявления боолее сложных сущностей.

Оператор typeof возвращает тип аргумента

ОСНОВНЫЕ ОПЕРАТОРЫ

Операнд – то, к чему применяется оператор
Унарным - называется оператор, который применяется к одному операнду
Бинарным - называется оператор, который применяется к двум операндам

ОПЕРАТОРЫ сравнения и логические значения
alert( '2' > 1 ); // true, сравнивается как 2 > 1
alert( '01' == 1 ); // true, сравнивается как 1 == 1
alert( false == 0 ); // true, false становится числом 0
alert( true == 1 ); // true, так как true становится числом 1.

ВЗАИМОДЕЙСТВИЕ с пользователем: alert, prompt, confirm
alert выводит сообщение.
prompt выводит сообщение и ждёт, пока пользователь введёт текст, а затем возвращает введённое значение или null, если ввод отменён (CANCEL/Esc).
confirm выводит сообщение и ждёт, пока пользователь нажмёт «OK» или «CANCEL» и возвращает true/false.
